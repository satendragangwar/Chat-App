# Chat-application

# To Run Locally

1- git clone <br/>
2 - cd client && npm install <br/>
3 - cd server && npm install <br/>
4 - Add mongoDd link <br/>

# Start

1 - npm run dev (server) <br/>
2 - npm start (client) <br/>

# Tech stack
Frontend - ReactJs
Backend -  Nodejs , mongodb , expressJs
Real-time-chat - Socket.io
